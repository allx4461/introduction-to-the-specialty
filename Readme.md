# Microservice Ascii Aquarium

## идея
интерактивный аски-аквариум, можно добавлять рыбов. можно читать оправдания, которые обновляются каждые 10 сек


## функциональность
* **Рендеринг аквариума:** Непрерывная генерация и анимация ASCII-графики с движущимися объектами (рыбы, водоросли).
* **Интерактивность:** Возможность динамически добавлять рыб в аквариум через UI (кнопки Fish 1, Fish 2, Fish 3).
* **Генератор оправданий:** Интеграция со сторонним API для получения отговорок, обновляется каждые 10 секунд. (получено из [отсюда](https://github.com/public-apis/public-apis#dictionaries))
 
## архитектура
1. **Frontend :** HTML/JS страница. Отвечает за опрос бэкендов (long-polling/setInterval) и обновление DOM-дерева (`innerHTML` для отрисовки графики).
2. **Backend 1 :** C++ сервер. Отвечает за логику координат, анимацию и отдачу "кадров" аквариума.
3. **Backend 2 :** Python (FastAPI) сервис. Выступает в роли информера, который ходит во внешний интернет за данными, форматирует их и отдает клиенту
